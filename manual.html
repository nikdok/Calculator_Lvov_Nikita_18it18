Руководство пользователя

После запуска приложения перед нами появится главное меню приложения на котором расположен калькулятор с необходимыми кнопками для выполнения арифметических операций.

  • Кнопки с цифрами, используемые для арифметических операций.
  • Кнопка точка - это знак, который ставится для вещественных чисел; Например: десятичных, сотенных, тысячных чисел и т.д.
  • Кнопки арифметических операций:
[+] - кнопка сложения чисел;
[-] - кнопка вычитания чисел;
[*] - кнопка умножения чисел;
[÷] - кнопка деления чисел;
[sin] - кнопка нахождения синуса числа;
[cos] - кнопка нахождения косинуса числа;
[tan] - кнопка нахождения тангенса числа;
[Х²] - кнопка возведение в квадрат числа;
[√] - кнопка вычисление квадратного корня числа;
  • Кнопка [С] - "сброс", т.е. сбрасывает всё поле ввода.
  • Кнопка [←] - "стереть символ", т.е. стирает 1 символ в поле ввода.
  • Кнопка [=] - вывод результата на дисплей.
  • Две скобки [ ) ] [ ( ] - используется для составления выражений.
